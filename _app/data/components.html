Nette Components

Nette is a full-stack PHP framework composed from a set of decoupled and reusable components. You can use any of these components standalone, independently from the rest of framework.

You can download each component from official Git repository. The recommended way how to install them is via Composer. For instance, to install the Latte use:

composer require latte/latte

Names of other components are in the third column in the table below.

Complete framework (i.e. all components) can be added using:

composer require nette/nette

List of Components

Application The kernel of web application nette/application
Bootstrap Bootstrap of your application nette/bootstrap
Caching Cache layer with set of storages nette/caching
Component Model Foundation for component systems nette/component-model
DI Dependency Injection Container nette/di
Database Database layer nette/database
Finder Finds files and directories with intuitive interface nette/finder
Forms Greatly facilitates secure web forms nette/forms
Http Layer for the HTTP request & response nette/http
Latte Amazing template engine latte/latte
Mail Sending E-mails nette/mail
Neon Loads and dumps NEON format nette/neon
Php Generator PHP code generator nette/php-generator
Reflection PHP class reflection enhancements nette/reflection
Robot Loader The most comfortable autoloading nette/robot-loader
Safe Stream Safe atomic operations with files nette/safe-stream
Security Provides access control system nette/security
Tester Enjoyable unit testing in PHP nette/tester
Tokenizer String tokenization nette/tokenizer
Tracy Debugging tool you will love ♥ tracy/tracy
Utils Utilities and Core Classes nette/utils
deprecated Deprecated stuff nette/deprecated
